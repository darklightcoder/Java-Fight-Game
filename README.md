# Java-Fight-Game
Just a simple fighting game prototype made using Gemini. Testing how fast you can develop a 2d game with AI.

To run issue the following commands in the terminal. Make sure you are inside project folder.
1. javac StreetFighterGame.java GamePanel.java Player.java
2. java StreetFighterGame

Keyboard Control Map 

Action	      Player 1 (Blue)	Player 2 (Red)
Walk  Left	  A	              Left Arrow
Walk  Right	  D	              Right Arrow
Jump	        W	              Up Arrow
Punch	        F	              I (or Numpad 1)
Kick        	G	              O (or Numpad 2)
